About

Manufacturing Quality Control Automation System

Solar QC Dashboard

An automated quality control pipeline for manufacturing solder point inspection. The system uses OCR and image processing to extract measurement data from factory test equipment, applies configurable QC rules, and generates detailed inspection reports. The latest intelligence layer adds historical retrieval, AI summaries, and operator feedback learning.

Image Processing

OpenCV-based preprocessing with deskewing, perspective correction, and CLAHE enhancement

OCR Engine

Google Cloud Vision OCR with confidence scoring and multi-pass extraction

Rule-Based QC

Three-tier quality criteria evaluation (Rule A, B, C) per manufacturing specification

Cross-Verification

Image vs Excel source comparison to validate OCR accuracy within configurable tolerance

Historical Intelligence

RAG-powered matching, local AI summaries, and operator feedback to learn from past batches

Built with FastAPI · Python · Vanilla JS · RAG + Local AI
